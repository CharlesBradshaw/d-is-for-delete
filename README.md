# d-is-for-delete
vim- easy mapping to make 'd' delete (not cut), and 'leader-d' cut.

Installation
============

There are 2 flavors to use this key mapping, either with a shared clipboard configuration or using the default unnamed register (.i.e '"' register).

Shared Clipboard:
-----------------

wget -O - >> file 

Default Register:
-----------------

wget -O - >> file
